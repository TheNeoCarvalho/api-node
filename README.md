# api-node

## Clonar repositório

```
git clone https://github.com/TheNeoCarvalho/api-node.git
```

## Instalar dependências

```
cd api-node
npm i
```

ou

```
cd api-node
yarn
```

## Alterar o nome do arquivo _.env-example_ para _.env_

<br>

## Rodar a aplicação

```
npm run dev
```

ou

```
yarn run dev
```

## Abrir o navegador e digitar

[http://localhost:3000](http://localhots:3000)
